## Prerequisites

- C++ compiler with C++17 support
- CMake 3.5 or higher
- Internet connection to fetch dependencies

## Installation

1. **Clone the repository**:
    ```sh
    git clone <https://github.com/tue09/Project_AI/edit/master/README.md>
    cd <build>
    ```

2. **Build the project**:
    ```sh
    mkdir build
    cd build
    cmake ..
    make
    ```
